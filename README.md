# BabelShips

A Battleships game written using BabelJS, Browserify and Gulp.

## Setup

Install Node and npm with [Homebrew](http://brew.sh/):

```
$ brew install node
```

Install Gulp globally:

```
$ npm install gulp -g
```

Install dependencies:

```
$ npm install
```

Build assets, run tests and launch in browser:

```
$ gulp
```
